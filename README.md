# bleepr-manage

Management system for bleepr

## Requirements

* Python 2.7
* libjpeg (for Pillow)

## Installation

* `pip install virtualenv`
* `virtualenv venv`
* `. venv/bin/activate`
* `git clone https://github.com:bleepr/bleepr-manage`
* `cd bleepr-manage`
* `pip install -r requirements.txt`


## Running the app

* `python -O app.py`
  - Or `python app.py` if you want to start in debug mode.
